Собираем проект:

$ bem make

Первая сборка с bem-tools займёт некоторое время, так как именно в этот момент в дирректорию проекта утанавливаются все необходимые npm-пакеты.

Для запуска сервера выполняем:

bem server

В результате получаем следующее сообщение:

inafo: Server is listening on port 8080. Point your browser to http://localhost:8080

Проблема?

Если порт 8080 уже используется другой программой, его можно переназначить с помощью аргумента -p
